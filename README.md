# Quality Degrader

## Overview

This is a *Factorio* mod that adds a machine that will downgrade any quality item placed into it by a single quality level.

This is for the early-mid game before you have unlocked the recycler. It gives you a way to handle an excess of intermediate quality items blocking upstream production, especially if you put quality modules in your mining drills and furnaces. Now you can achieve an all-rare Nauvis factory and space platforms before ever stepping foot on Fulgura!

## Features

- Adds the degrader, a 2x2 burner furnace for running quality degrading recipes, modeled on the burner mining drill.
- All recyclable items also get a quality degrading recipe to decrease quality by one level.
- All technologies that unlock quality are amended to also unlock the degrader.
